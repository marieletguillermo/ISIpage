Our Services

License Plate Recognition

The LPR application can be used for vehicle identification and tagging. It includes the following features:


  • Automated detection of license plates in vehicle images.
  • Automated translation of characters from license plate images into text.
  • Automated record and report generation of detected license plates based on location, date, and time.

People Detection

The people counting application can be used for monitoring pedestrian foot traffic. It includes the following features:


  • Automated detection and counting of people/pedestrian passing in CCTV coverage area.
  • Automated record and report generation of people volume and density based on location, date, and time.

Vehicle Classification and Counting

The vehicle counting/classification application can be used for traffic monitoring, congestion analysis, and transport/traffic planning. It includes the following features:


  • Automated detection and counting of vehicles passing in CCTV coverage areas.
  • Vehicle classification that can have customized labeling information based on client use-case.
  • Automated record and report generation of vehicle volume and density based on vehicle types, location, date, and time.
